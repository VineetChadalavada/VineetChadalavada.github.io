My research focuses on analyzing DDR4 timing side channels in multi-tenant FPGA cloud environments and developing real-time monitoring systems like the Trusted Memory Access Monitor (TMAM).

Selected Projects

Prime+Probe Attack on DDR4

Developed a DDR4-based timing side-channel attack on Xilinx Alveo U250. Measured row buffer timing variations to infer victim memory access patterns.

Trusted Memory Access Monitor (TMAM)

Designed a hardware IP for detecting DDR4-based side-channel attacks in real-time, integrated into the FPGA shell layer to detect malicious tenants.

RO-PUF Secure Boot System

Implemented a Ring Oscillator PUF on Zybo Z7 FPGA, improving response stability by 30% and enabling dynamic reconfiguration-based authentication.

AES Power Trace Analysis

Collected 20K power traces from CW305 Artix-7 FPGA and used CNN and SVM models to recover AES subkeys with high prediction accuracy.
